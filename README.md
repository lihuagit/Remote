# Remote
使用QT5.12.11+vs2019完成的一个小项目

## 实现功能
功能相对来说比较少，只实现了远程桌面查看控制，没有实现文件传输，写了一部分在代码里，但是还有bug没有修改完
客户端和服务端都在一个程序里，不是很稳定，建议测试的时候关闭程序切换客户端 服务端

### 远程桌面查看
这部分比较简单，使用QT带的函数抓取函数之后使用winsocket的TCP/IP协议传输，在客户端显示即可
